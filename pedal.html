Signal Pedal

A compact signal tool exploring tactile control, sonic transformation, and interface clarity.

Applied Study / Prototype Direction

Overview

This project explores the guitar pedal as a precise interface for shaping sound. Rather than adding complexity, the aim is to reduce the signal path to a small number of deliberate controls and to express transformation in a form that is tactile, legible, and musically useful.

Brief

Develop a pedal concept that demonstrates KUON Studio’s approach at signal scale: reduced form, careful interaction, and a clear relationship between physical control and sonic behaviour.

Design Intent

The project focuses on compactness, usability, and material precision. It is intended not only as a product opportunity, but also as a positioning object that reveals how the studio thinks about sound, transformation, and use.

Within KUON Studio

This project sits within the studio’s work on signal, examining how sound can be shaped through reduced interfaces and physically legible objects. It connects the performance logic of the guitar to wider work in objects, space, and listening.

What this project demonstrates

  • Signal-scale design thinking
  • Interface reduction and clarity
  • Translation of sonic behaviour into product form
